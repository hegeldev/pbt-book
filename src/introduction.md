# Introduction

This is a book about **property-based testing**. It is also a book about Hegel,
our family of property-based testing libraries for a variety of languages.

We assume you're familiar with software testing but not necessarily property-based
testing. If you've used another property-based testing library, we expect this book
will still be useful for you (and not just for the Hegel content), but much of it
will be skippable.
