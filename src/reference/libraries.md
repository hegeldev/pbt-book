# Libraries

Hegel provides official, fully-fledged property-based testing libraries for
several languages, all built on the shared [hegel-core](https://github.com/hegeldev/hegel-core)
server:

- [hegel-rust](https://github.com/hegeldev/hegel-rust)
- [hegel-go](https://github.com/hegeldev/hegel-go)
- [hegel-cpp](https://github.com/hegeldev/hegel-cpp)
- [hegel-typescript](https://github.com/hegeldev/hegel-typescript)
- [hegel-core](https://github.com/hegeldev/hegel-core) — the server

Community-maintained libraries are listed on the [Community](../project/community.md)
page.
