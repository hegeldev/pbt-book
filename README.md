# Property-based testing with Hegel

Source for a [_Property-based testing with Hegel_](https://hegel.dev) book, an
[mdBook](https://rust-lang.github.io/mdBook/) about [Hegel](https://github.com/hegeldev).

This book is currently *extremely* early access. It is being written in public, and we hope
that parts of it are already useful to readers, but it is by no means in a polished state.
